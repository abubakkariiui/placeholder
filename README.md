# PlaceholderJS
A simple image placeholder for JavaScript.


## Usage
```npm i placeholder```

```yarn add placeholder```

```pnpm add placeholder```



### NextJS Example

```
import { Placeholder } from "placeholder"; // Import the NextJS image component

<Placeholder width="500" height="500" />
```
